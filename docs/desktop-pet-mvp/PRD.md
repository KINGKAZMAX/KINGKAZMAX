# 桌面级猫咪宠物 MVP PRD

- 版本：v0.1
- 日期：2026-05-09
- 负责人：产品 / 设计 / 客户端 / 后端 / AI
- 目标平台：Windows（首发）

## 1. 背景与目标

### 1.1 背景
用户希望上传自家猫咪，生成可在桌面互动的宠物角色，用于情感陪伴与轻量提醒。

### 1.2 MVP 目标
1. 用户可上传猫咪图片并生成专属宠物。
2. 宠物可在桌面执行基础动作与点击互动。
3. 提供喝水/久坐提醒与全屏静默。
4. 验证 D1、D7 留存和互动频次。

### 1.3 成功指标
- 宠物生成成功率 ≥ 95%
- 平均生成耗时 ≤ 30 秒
- D1 留存 ≥ 45%
- D7 留存 ≥ 20%
- 人均日互动次数 ≥ 8

## 2. 功能范围

### 2.1 P0（MVP）
- 登录或游客进入。
- 上传 1~3 张猫咪图片（JPG/PNG，单张 ≤ 10MB）。
- 创建生成任务并返回进度。
- 桌面透明窗口展示宠物（置顶、拖拽、边缘吸附）。
- 基础动作：行走、坐下、睡觉、短时跟随鼠标。
- 互动动作：摸头、喂食。
- 提醒：喝水、久坐。
- 全屏/会议时自动隐藏或静默。

### 2.2 P1（后续）
- 多宠物切换。
- 配件系统（项圈、帽子等）。
- 截图分享。

### 2.3 本期不做
- 高精度 3D 宠物生成。
- 语音聊天。
- 用户自定义动作编辑器。

## 3. 用户流程
1. 首次启动进入欢迎页。
2. 上传猫咪照片并触发生成。
3. 进度页显示任务状态。
4. 生成成功后命名宠物并进入桌面驻留。
5. 用户进行互动与提醒设置。

## 4. 非功能需求
- 客户端内存常驻 < 250MB。
- 空闲 CPU 占用 < 5%。
- 崩溃率 < 0.5%。
- 端到端可用性 ≥ 99%。

## 5. 埋点事件
- upload_start / upload_success / upload_fail
- generate_start / generate_success / generate_fail
- pet_interaction_click
- reminder_shown / reminder_clicked / reminder_closed
- auto_hide_fullscreen_triggered

## 6. 验收标准
- 90% 样本图片可成功生成。
- 生成完成后 10 秒内完成桌面展示。
- 全屏隐藏准确率 100%。
- 提醒开关实时生效。
